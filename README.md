# Nuxt 3

## Version Specifications

- Node.js: 18.13.0
- Dependencies Manager: pnpm 7.25.1
- Nuxt: 3.0.0

## Getting Started

If you don't have pnpm installed yet, just [click here](https://pnpm.io/installation) to install it.

### Install Dependencies

```bash
pnpm install
```

### Development

```bash
pnpm dev
```

### Production

```bash
pnpm start
```
